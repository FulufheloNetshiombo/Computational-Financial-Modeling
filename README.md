# Computational-Financial-Modeling
📊 Computational Financial Modeling with SQL

BrightLearn Practical Series – Assessment BL-DS-FIN-02

📌 Project Overview

This project demonstrates how financial mathematics can be implemented using SQL in Microsoft SQL Server Management Studio.

The goal is to translate financial formulas into SQL queries and produce accurate, reliable outputs formatted to two decimal places (ZAR).

It focuses on combining data engineering and financial modeling skills to solve real-world financial calculations.

🎯 Objectives

This project covers the following key skills:

Algorithmic Implementation
Writing SQL queries that model financial calculations correctly
Mathematical Translation
Converting formulas into SQL expressions
Numerical Precision
Using DECIMAL data types to ensure accurate financial reporting
Model Validation
Matching outputs with predefined reference results
🛠️ Technical Environment
Tool: SQL Server Management Studio
Database: FinancialEngineeringDB
Table: FinancialTasks
Language: SQL
